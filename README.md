# oxymetre
A3 project at ISEN Brest

## Installation

Clone the repository

```bash
git clone https://github.com/ClementYziquel5/oxymetre
```

## Usage

```bash
cd oxymetre
make
./main.exe
```

Some test files are available (record1_xx.dat), you can select them in main.c
